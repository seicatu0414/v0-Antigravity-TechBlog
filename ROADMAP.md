This document outlines the planned features and development roadmap for the project.

## 管理者機能

### 管理ダッシュボード
**URL**: `/admin`

**機能**:
- ユーザー統計 (総数、管理者数)
- 記事統計 (総数、公開数)
- タグ統計
- コメント統計
- アクティビティログ (10件ずつ表示、さらに表示ボタン)

### ユーザー管理
**URL**: `/admin/users`

**機能**:
- ユーザー一覧
- ユーザー詳細
- ユーザー新規登録
- ロール変更 (一般 / 管理者)
- ユーザー削除

### タグ管理
**URL**: `/admin/tags`

**機能**:
- タグ一覧
- タグ作成
- タグ編集
- タグ削除
- アクティビティログ記録

### ヒーロー画像管理
**URL**: `/admin/hero-images`

**機能**:
- ヒーロースライド一覧
- スライド追加
- スライド編集
- スライド削除
- 並び順変更
